# zapret-youtube-discord-roblox
Данный репозиторий был основан на https://github.com/Flowseal/zapret-discord-youtube.
В данной версии zapret (оригинал выше) имеются вписанные ссылки и IP адресса серверов компаний "Roblox Corporation" и "Amazon Web Services" в следующих файлах: ipset-all.txt, list-general; позволяющие обходить блокировку со стороны Роскомнадзора (далее — РКН).
